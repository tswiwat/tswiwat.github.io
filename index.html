เริ่มต้นเรียนรู้การเขียนโปรแกรมคอมพิวเตอร์เบื้องต้น

บทเรียนออนไลน์เชิงโต้ตอบแบบอินเตอร์แอคทีฟวิชาการเขียนโปรแกรมคอมพิวเตอร์เบื้องต้น (Python Programming) ออกแบบสำหรับนักศึกษาปริญญาตรี ชั้นปีที่ 1

โครงสร้างหลักสูตร (Curriculum)

1

พื้นฐานและโครงสร้างควบคุม

ตัวแปร, ชนิดข้อมูล, ตัวดำเนินการคณิตศาสตร์/ตรรกศาสตร์, เงื่อนไข if-else, ลูปวนซ้ำ for/while

2

โครงสร้างข้อมูล (Data Structures)

การจัดการรายการข้อมูลแบบเรียงลำดับและแบบคีย์-ค่า ด้วย List, Tuple, และ Dictionary

3

การใช้งานฟังก์ชัน (Functions)

การจัดกลุ่มโค้ดทำงานด้วย def, การรับพารามิเตอร์และการรีเทิร์นค่ากลับไปใช้ต่อ

4

การจัดการไฟล์และการทำงานระดับสูง

การเขียนอ่านไฟล์ภายนอก, ป้องกันโปรแกรมพังด้วยการดักจับข้อผิดพลาด try-except

ไฮไลท์พิเศษของระบบการเรียนรู้

  • รันโค้ด Python ในหน้าเว็บได้ทันที:

    ไม่ต้องติดตั้งโปรแกรมเสริมใดๆ เขียนโค้ดใน Sandbox และทดลองกด Run เพื่อดูผลลัพธ์ผ่านบราวเซอร์ได้เลย

  • แบบทดสอบ Bloom's Taxonomy:

    ท้าทายตนเองด้วยแบบทดสอบ 3 ระดับ ทั้งแบบวัดความเข้าใจ, เติมคำทำนายผลลัพธ์, และโจทย์เขียนโค้ดแก้ปัญหา

  • บันทึกความก้าวหน้าอัตโนมัติ:

    ระบบจดจำความก้าวหน้าของคุณและการทำแบบทดสอบผ่านระบบ Local Storage กลับมาเรียนเมื่อไหร่ก็ต่อจุดเดิมได้ทันที
